# EmberOS Prompt Library

This folder stores the prompt specifications used by EmberOS.

Prompts are part of the product system and must be version controlled.

## Rule

Prompts must follow the EmberOS V1 Blueprint and Decision Log.

Do not create prompts that bypass locked product decisions.
