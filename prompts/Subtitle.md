# Subtitle Prompt Specification

## Purpose

Generate viewer-facing subtitles for the final marketing video.

Subtitles are for the audience, not for the business owner.

## Input

- Content Analysis
- Marketing Angle
- Video Planning
- Subtitle Language
- Output Language

## Output

Structured subtitle segments with:

- Start time
- End time
- Primary subtitle line
- Secondary subtitle line when bilingual mode is enabled
- Translation notes if needed

## Rules

- Subtitles must not come directly from Business Strategy.
- Subtitles must be natural and audience-facing.
- Do not use generic terms such as brand awareness as on-video text.
- Support bilingual subtitles for V1.
- Translation should be marketing translation, not literal translation.
- Default style should be premium minimal, stable, non-bouncy and professional.

## V1 Scope

Subtitles should help viewers understand the video and message quickly.
