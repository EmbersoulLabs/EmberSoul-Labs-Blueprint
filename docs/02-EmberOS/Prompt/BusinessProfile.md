# Business Profile Prompt

## Purpose

Generate and validate Business Profile context for EmberOS AI workflow.

## Input

- Company Name
- Industry
- Services
- Region
- Target Audience
- Brand Voice

## Output

- Clean Business Profile Context
- Missing Information
- AI Confidence
- Suggested Confirmation Questions

## Rules

- Do not invent business facts.
- If information is missing, ask for confirmation.
- Use Business Profile as long-term context.
- Do not override Campaign-specific context.
