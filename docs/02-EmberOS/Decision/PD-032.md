# PD-032 - Target Audience Inference

Status:
Locked

Decision:
Target Audience should be AI-inferred first, then user-editable.

Rules:
- AI infers Target Audience from Industry, Services, and Region.
- User may modify the result.
- User-confirmed Target Audience becomes the source of truth.

Reason:
Many SME owners do not clearly know how to define their target audience. EmberOS should help infer it while allowing user correction.
